# BarrierFree Map · 배리어프리 지도

> **"시설이 있다는 것보다 중요한 건, 지금 실제로 이동할 수 있는지입니다."**
>
> 공공데이터는 시설의 존재를, 커뮤니티는 실제 이용 가능 여부를 알려줍니다.

교통약자(휠체어 이용자·유아차 사용자·노약자·시각장애인)를 위한 **실시간 배리어프리 경로 안내 및 접근성 정보 공유 웹앱**입니다. 공공 API 기반 정적 시설 정보와 사용자 제보 기반 실시간 이동 가능성 정보를 결합합니다.

## 주요 기능

- 🗺️ **지도** — 공공시설 마커 + 사용자 제보 마커 + 커뮤니티 글 마커, 이동 모드별 캐릭터 아바타, 카테고리 필터
- 🧭 **경로** — 교통약자 맞춤 경로 추천(추천·빠름·안전), 제보 회피 반영
- 👥 **커뮤니티** — 장소 기반 이동 경험 공유 피드(실시간 제보·시설 상태·후기·질문·해결 알림), 댓글
- 🏢 **시설** — 공공 데이터 + 실이용 후기 + 최근 커뮤니티 글
- 👤 **마이** — 기여도·뱃지·아바타 꾸미기 + 지역 접근성 리포트(B2G·B2B 대시보드)

## 기술 스택

- React 18 + TypeScript + Vite
- Tailwind CSS
- Zustand (전역 상태)
- Recharts (대시보드 차트)

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 프로덕션 빌드
```

## 공공 API 연동

`src/services/publicAccessibilityApi.ts` 가 진입점이며, 현재는 mock 데이터를 반환합니다.
`VITE_USE_REAL_API=true` 환경변수와 실제 엔드포인트를 구현하면 실 API로 교체할 수 있습니다.

> MVP 지역: 고려대학교 안암 캠퍼스 일대 (안암역·고대안암병원·참살이길·개운산공원 등)
