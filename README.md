# BarrierFree Map · 배리어프리 지도

> **"시설이 있다는 것보다 중요한 건, 지금 실제로 이동할 수 있는지입니다."**
>
> 공공데이터는 시설의 존재를, 커뮤니티는 실제 이용 가능 여부를 알려줍니다.

**교통약자를 위한 실시간 배리어프리 지도와 보호자 안심 공유 플랫폼**입니다. 휠체어 이용자·유모차 사용자·노약자·시각장애인·임산부 등 이동 약자가 "지금 이동할 수 있는지"를 확인하고, 필요할 때 보호자와 안전하게 이동 상태를 공유할 수 있도록 돕습니다. 공공 API 기반 정적 시설 정보와 사용자 제보 기반 실시간 이동 가능성 정보를 결합합니다.

## 개발 단계

현재 불편지도는 웹 기반 기능형 프로토타입 개발 완료 단계입니다. React, TypeScript, Vite 기반으로 교통약자 유형별 지도, 접근성 시설 정보, 실시간 통행 불편 제보, 커뮤니티, 맞춤 경로 추천 시뮬레이션, 접근성 리포트 대시보드를 구현했습니다. 현재는 고려대학교 안암캠퍼스·안암역·고대안암병원 주변 Mock 데이터를 기반으로 핵심 사용자 흐름을 검증하고 있으며, 향후 실제 지도 API, 공공데이터 API, 서버 DB, 사용자 인증, 사진 제보, 보호자 공유 모드를 연동해 실사용 MVP로 고도화할 계획입니다.

> 🧪 **현재 MVP 테스트 지역: 고려대학교 안암캠퍼스 · 안암역 · 고대안암병원 주변**
> 전국 서비스가 아니라, 특정 생활권에서 핵심 사용자 흐름을 검증 중인 프로토타입입니다.

## 주요 기능

- 🗺️ **지도** — 공공시설 마커 + 사용자 제보 마커 + 커뮤니티 글 마커, 이동 모드별 캐릭터 아바타, 카테고리 필터. 사용자 모드(휠체어·유모차·노약자·시각장애인·임산부)에 따라 강조 정보가 달라집니다.
- 🧭 **경로** — 교통약자 맞춤 경로 추천(추천·빠름·안전), 제보 회피 반영
- 🛡️ **안심 공유** — 보호자와 함께 안전하게 이동. 위치 공유 시작/중단, 도착·경로 이탈·장시간 정지·위험 구간 알림, 응급 알림, 개인정보 동의 흐름 (Mock)
- 👥 **커뮤니티** — 장소 기반 이동 경험 공유 피드(실시간 제보·시설 상태·후기·질문·해결 알림 + 보호자 질문·병원 동행 후기·부모님 이동 경로·유모차 동행 팁), 댓글
- 🏢 **시설** — 공공 데이터 + 실이용 후기 + 최근 커뮤니티 글
- 👤 **마이** — 기여도·뱃지·아바타 꾸미기 + 지역 접근성 리포트(반복 제보 구역·행정 연계 후보 포함, B2G·B2B 대시보드)

### 제보 신뢰도 시스템

각 제보에는 신뢰도(높음/보통/낮음), 확인 수, 사진 여부, 최신성, 상태(활성·확인 필요·만료 예정·해결됨·반박 있음)가 함께 표시됩니다. 오래되거나 확인이 필요한 제보에는 "아직 불편해요 / 해결됐어요 / 잘 모르겠어요" 재확인 버튼이 제공되어, 정보가 지금도 유효한지 함께 확인합니다.

### 보조 정보: 간접흡연 주의 구역

간접흡연 주의 구역은 서비스의 핵심 기능이 아니라, 임산부·유모차 사용자의 이동 선택을 돕기 위한 **보조 정보**입니다. 따라서 전체 사용자에게 항상 노출하지 않고 **임산부 모드·유모차 모드에서만** 지도에 표시됩니다.

## 기술 스택

- React 18 + TypeScript + Vite
- Tailwind CSS
- Zustand (전역 상태)
- Recharts (대시보드 차트)

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 프로덕션 빌드
```

## 공공 API 연동

`src/services/publicAccessibilityApi.ts` 가 진입점이며, 현재는 mock 데이터를 반환합니다.
`VITE_USE_REAL_API=true` 환경변수와 실제 엔드포인트를 구현하면 실 API로 교체할 수 있습니다.

> MVP 지역: 고려대학교 안암 캠퍼스 일대 (안암역·고대안암병원·참살이길·개운산공원 등)
